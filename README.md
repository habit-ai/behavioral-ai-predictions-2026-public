# Behavioral AI Predictions 2026

A prediction tournament comparing human forecasters against AI models on 40 predictions about AI developments in 2026.

## What This Is

This repository tracks weekly evidence and progress toward resolving 40 predictions about AI in 2026, spanning:

- **AI Industry** - IPOs, user growth, hardware launches
- **Benchmarks** - Model performance on LiveBench, FrontierMath, etc.
- **AI Safety** - Liability cases, misinformation, catastrophic risks
- **Research** - Studies on AI's psychological/behavioral impacts
- **AI Capabilities** - Autonomous agents, creative AI, task performance

## Weekly Reports

Each week, an automated system searches for news relevant to each prediction and generates a summary of notable developments.

Browse the [weekly reports](reports/weekly/) to see what's happening.
